# Installation

## 1. Using uv (Recommended)
```bash
pip install uv

cd your_path_to/SCALED-Particle
uv sync
souce .venv/bin/activate
uv pip install -e .

```

## 2. Using Conda

```bash
conda create -n scaled python=3.10
conda activate scaled

cd your_path_to/SCALED-Particle
pip install -e .

```
